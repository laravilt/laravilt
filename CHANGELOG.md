## 0.1.0 - 2025-01-10

### Added
- Initial release
- Meta-package for easy installation (includes all 14 core packages)
- Automatic package discovery and registration
- `laravilt:install` command for quick setup
- `laravilt:resource` command for resource generation
- Resource stub templates with full scaffolding
- Configuration file with panel, generation, and AI/MCP settings
- Complete installation guide and documentation
- Directory structure auto-creation (app/Laravilt/Resources, Pages)
- Example resource generation option
